# to study Gauche

## REPL

`gosh` or `rlwrap gosh`

exit: `(exit)`

## Run

`gosh xxx.scm`

## Gauche ドキュメント

### Chrome

- `chrome://settings/searchEngines` を開いて、
- gosh: `http://practical-scheme.net/gauche/man/?l=jp&p=%s` を設定。


## Pane

- `C-x 2` : 上下分割。
- `C-x 3` : 左右分割。
- `C-x 1` : 分割を消す。
- `C-x o` : 別のウィンドウを選択。
	- `C-c h|j|k|l` : 移動。
